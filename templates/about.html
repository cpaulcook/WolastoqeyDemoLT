About

This app was developed by Paul Cook as part of an assignment for ABRG3686 Wolastoqey Language 1. The source code is freely available.

This app follows the number system as presented in ABRG3686. There are some differences with the information presented on the Passamaquoddy-Maliseet Language Portal on the number system (e.g., 'cel vs cel).

Feedback on this app would be very welcome. If you spot errors, please let me know. Woliwon!
